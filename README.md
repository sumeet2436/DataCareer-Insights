# DataCareer Insights

## Overview

DataCareer Insights is an end-to-end Data Engineering project designed to analyze global Data Science and Data Engineering job market trends.

The platform processes salary data, stores it in PostgreSQL, performs analytical queries, and visualizes insights using Power BI.

---

## Problem Statement

Job seekers often struggle to understand:

* Which roles pay the highest salaries
* Which countries offer better compensation
* How experience affects salary
* Remote vs onsite opportunities

This project provides data-driven insights into these questions.
